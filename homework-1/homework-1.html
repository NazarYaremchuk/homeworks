Основні команди Git:

  • git config --global user.name - показує ім'я користувача
  • git config --global user.email - показує email користувача
  • git init - запуск репозиторів у поточній папці
  • git status - показує поточний статус
  • git add - додає усі файли
  • git commit- збереження змін у commit
  • git log- показує список усіх commit (від найновішого)
  • git checkout - перемикається на іншу гілку
  • git branch - робота з гілками в репозиторії
  • git push - заливає поточні локальні commits на віддалений репозиторій
  • git pull - забирає зміни з віддаленого репозиторія в локальний
  • git merge - об'єднує гілки
  • git clone - файл не коммітиться і залишається в локальному репозиторії

Що таке gitignore файл

.gitignore файл - потрібен для приховування файлів та папок від системи контролю версій Git.

Домашнє завдання "Advanced"

  • git revert - відміна операції без втрати історії коммітів
  • git reset - відміна зміни у локальному репозиторії
  • git rebase - перенесення змін з однієї гілки на іншу
  • git cherry-pick - перенесення конкретних коммітів з одного місця репозиторія в інший
  • git stash - відкладення змін та їх внесення пізніше
